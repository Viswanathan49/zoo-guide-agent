# Zoo Tour Guide AI Agent 🦒

A multi-agent AI app built with Google ADK and Gemini 2.5 Flash.

## Tech Stack
- Google ADK
- Gemini 2.5 Flash
- LangChain Wikipedia Tool
- Google Cloud Run
- Python

## How it works
1. User asks a question about zoo animals
2. Researcher Agent searches Wikipedia
3. Formatter Agent creates a friendly response
